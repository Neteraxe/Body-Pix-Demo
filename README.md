# 具体内容

本工程是基于body-pix的人物分割模型生成语义分割图。不含npm的前端工程。

# 概要设计

集成了body-pix实现语义分割图的功能，并使用input和button实现了拖拽图片和上传图片。
JavaScript依赖库：bootstrap, jquery, tensorflow/tfjs, tensorflow-models/body-pix.
